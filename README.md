# Книгарня

Простий проєкт книгарні на Python, який дозволяє керувати книгами, клієнтами та базовими операціями магазину.

**Проєкт підтримує Python ≥ 3.12.**

## Функції
- Перегляд каталогу: відображення всіх доступних книг, відсортованих за категоріями, жанрами, авторами або видавництвами.
- Детальна сторінка книги: надання вичерпної інформації про книгу (назва, автор, видавництво, рік, опис, ціна, обкладинка, рейтинг, наявність).
- Пошук: швидкий пошук книг за назвою, автором, ISBN або ключовими словами.
- Фільтрація та сортування: можливість фільтрувати книги (наприклад, за ціновим діапазоном, жанром) та сортувати їх (за популярністю, датою додавання, ціною).
- Реєстрація та вхід: можливість створення нового облікового запису та входу для здійснення покупок.
- Профіль користувача: перегляд та редагування особистих даних, адреси доставки.
- Історія замовлень: перегляд усіх попередніх замовлень, їхнього статусу та деталей.
  
## Як запустити проєкт

### 1. Клонування репозиторію
```bash
git clone <URL_репозиторію>
cd <назва_папки_проєкту>
```

### 2. Створення віртуального середовища
```bash
python3 -m venv venv
```
```bash
source venv/bin/activate   # Mac/Linux
# venv\Scripts\activate    # Windows
```

### 3. Встановлення залежностей
```bash
pip install -r requirements.txt
```

### 4. Налаштування бази даних
```bash
python manage.py migrate
```

### 5. Створення суперкористувача (адмін)
Щоб мати доступ до адміністративної панелі:
```bash
python manage.py createsuperuser
```

### 6. Запуск сервера Django
```bash
python manage.py runserver
```
Бекенд буде доступний за адресою: http://127.0.0.1:8000/
Адмінка: http://127.0.0.1:8000/admin/

### 7. Запуск тестів
```bash
python manage.py test
```
